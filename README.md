Gold Price Analysis 

Historical analysis of the previous decade's gold pricing trends. 
I did this project as I wanted to apply the statistical programming skills I've been learning in my masters program onto a small, real-world and commodity-based dataset.

The data was sourced from Kaggle: https://www.kaggle.com/datasets/abdulmaliklodhra/gold-price-dataset-2016-2026
Key Features: Highs, lows, mean, plot and insights

This project is currently in progress, I will be performing further statistical inference on the dataset in future.
