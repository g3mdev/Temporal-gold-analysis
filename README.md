Temporal Gold Analysis 

This project analyses historical gold prices over a 10 year period to explore trends, fluctuations and yearly extremes. Using Python (Pandas, Matplotlib), I applied OOP principles to calculate annual highs, lows and means and visualised them for clarity.
Dataset: Gold Price Dataset 2016-2026 — sourced from Kaggle: https://www.kaggle.com/datasets/abdulmaliklodhra/gold-price-dataset-2016-2026

Key Features:

*Annual highs, lows and mean calculations

*Price trend visualisation

*Statistical insights across the decade

This project is currently in progress — further statistical inference will be added in future.
